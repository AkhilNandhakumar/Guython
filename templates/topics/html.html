HyperText Markup Language

Also known as HTML, this is the programming language widely used to display information, such as images, text, and more on website pages.

{#
#}

HTML is the basis of frontend web programming. It holds all of the raw visuals seen on a website, without formatting. CSS complements HTML by presenting it in a pleasing and organized manner. An analogy to help understand would be like Google Slides. HTML would compare with text and images on the slides, and CSS would compare with animations, transitions, font color, effects, sizing, organization, etc.
