Physics Class Experience

In Physics, you complete a lot of worksheets that are given by the teacher. The teacher usually goes over the answer to the worksheets the next day in class. The students learn a lot about certain formulas that allow them to calculate the speed, velocity, and acceleration of objects. There a multiple units where students learn new topics using different formulas.

Formulas Math
In Physics you are given a bunch of formulas, you need to be able to recognize which formula is appropriate for the given problem. You need to be able to make the proper calculations for all the formulas, since the answers could be related to each other. You need to be able to make simple math calculations such as multiplication, addition, subtraction and division. Knowing these will make the class very easy to follow and understand.
